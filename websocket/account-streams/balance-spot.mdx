## Query spot balance (one-time)

Fetch current spot balances for specific assets without subscribing.

**Request**

```json
{
  "id": 2,
  "method": "balanceSpot_request",
  "params": ["ETH", "BTC"]
}
```

Pass an empty array to query all assets.

**Response**

```json
{
  "id": 2,
  "result": {
    "ETH": {
      "available": "0",
      "freeze": "0"
    },
    "BTC": {
      "available": "0",
      "freeze": "0"
    }
  },
  "error": null
}
```
## Subscribe to spot balance

<Steps>
  <Step title="Send subscription request">
    Pass asset tickers in `params`. Pass an empty array to subscribe to all assets.

    ```json
    {
      "id": 3,
      "method": "balanceSpot_subscribe",
      "params": ["USDT", "ETH"]
    }
    ```
  </Step>
  <Step title="Receive confirmation">
    ```json
    {
      "id": 3,
      "result": { "status": "success" },
      "error": null
    }
    ```
  </Step>
  <Step title="Receive real-time updates">
    The server pushes `balanceSpot_update` when any subscribed asset balance changes.

    ```json
    {
      "id": null,
      "method": "balanceSpot_update",
      "params": [
        {
          "USDT": {
            "available": "100.1885",
            "freeze": "0"
          }
        }
      ]
    }
    ```

    - `available` — amount free to trade
    - `freeze` — amount locked in active orders
  </Step>
  <Step title="Unsubscribe">
    ```json
    {
      "id": 4,
      "method": "balanceSpot_unsubscribe",
      "params": []
    }
    ```
  </Step>
</Steps>
