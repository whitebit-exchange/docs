## Ping

Send a ping every 50 seconds to keep the connection alive. The server closes the connection after 60 seconds of inactivity.

**Request**

```json
{
  "id": 0,
  "method": "ping",
  "params": []
}
```

**Response**

```json
{
  "id": 0,
  "result": "pong",
  "error": null
}
```

## Server Time

**Request**

```json
{
  "id": 1,
  "method": "time",
  "params": []
}
```

**Response**

```json
{
  "id": 1,
  "result": 1493285895,
  "error": null
}
```

`result` is the current server time as a Unix timestamp (integer seconds).
