Market statistics for the current UTC day, resetting daily at 00:00 UTC. **Update interval:** 1 second.

## Query market today (one-time)

<Note>
The query method accepts **only one market per request**. To get data for multiple markets, either make separate requests or use the subscription method.
</Note>

**Request**

```json
{
  "id": 1,
  "method": "marketToday_query",
  "params": ["BTC_USDT"]
}
```

**Response**

```json
{
  "id": 1,
  "result": {
    "last": "105000.00",
    "open": "101000.00",
    "high": "106000.00",
    "low": "100500.00",
    "volume": "1234.56789",
    "deal": "129876543.21"
  },
  "error": null
}
```

## Subscribe to market today

<Steps>
  <Step title="Send subscription request">
    Pass one or more market names in `params`.

    ```json
    {
      "id": 1,
      "method": "marketToday_subscribe",
      "params": ["BTC_USDT", "ETH_USDT"]
    }
    ```
  </Step>
  <Step title="Receive confirmation">
    ```json
    {
      "id": 1,
      "result": { "status": "success" },
      "error": null
    }
    ```
  </Step>
  <Step title="Receive real-time updates">
    The server pushes `marketToday_update` every second.

    ```json
    {
      "id": null,
      "method": "marketToday_update",
      "params": [
        "BTC_USDT",
        {
          "last": "105000.00",
          "open": "101000.00",
          "high": "106000.00",
          "low": "100500.00",
          "volume": "1234.56789",
          "deal": "129876543.21"
        }
      ]
    }
    ```

    `params[0]` is the market name. `params[1]` is the statistics object.
  </Step>
  <Step title="Unsubscribe">
    ```json
    {
      "id": 2,
      "method": "marketToday_unsubscribe",
      "params": []
    }
    ```
  </Step>
</Steps>

## Field descriptions

| Field | Description |
|-------|-------------|
| `last` | Current last trade price |
| `open` | Opening price at today's UTC midnight |
| `high` | Highest price today |
| `low` | Lowest price today |
| `volume` | Total trading volume in [stock](/glossary#stock) currency |
| `deal` | Total trading volume in [money](/glossary#money) currency |

<Note>
All statistics reset to zero at UTC midnight (00:00 UTC) each day.
</Note>
